# cd-vue-starter
### VueJS Starter


### Getting Started

Install Repo

```sh
$ git clone https://github.com/mikeerickson/cd-vue-starter.git
```

Install dependencies

```
$ npm i
```

Start Development Server
```
$ npm start
```
