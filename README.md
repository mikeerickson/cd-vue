# cd-vue-starter
### VueJS Starter


View on [npm](https://www.npmjs.com/package/cd-vue)


### Install with [npm](npmjs.org)

```sh
npm i -S cd-vue
```

### Install with [yarn](https://yarnpkg.com/lang/en/docs/install/#mac-tab)

```sh
yarn add cd-vue
```
